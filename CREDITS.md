# Credits

## episeerr

The hold tag + delay profile pattern used in CTRLr's episode picker (applying a
temporary Sonarr delay profile to prevent grabs while episode monitoring is
configured) was inspired by [episeerr](https://github.com/Vansmak/episeerr) by
**Vansmak**.

episeerr is a standalone tool that adds episode-level granular control,
viewing automation, and storage management to Sonarr via Overseerr/Jellyseerr.
If you need a full-featured solution for episode automation, check it out.
